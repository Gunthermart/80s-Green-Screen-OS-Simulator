🇫🇷 Français+

Objectif: Détruisez tous les chars jaunes. Évitez de vous écraser.
Drone/Hélico PC: Flèches pour altitude/rotation. Souris pour direction. Espace pour bombe, F pour missile.
Avion PC: Flèches Haut/Bas pour les gaz. Flèches Gauche/Droite pour le lacet. Souris pour tangage/roulis.
Contrôles Mobiles: Joystick gauche pour altitude/rotation (gaz/lacet pour avion). Joystick droit pour direction (tangage/roulis pour avion). Bouton (B) pour bombe, (M) pour missile.

🇬🇧 English+

Objective: Destroy all yellow tanks. Avoid crashing.
Drone/Heli PC: Arrow Keys for altitude/yaw. Mouse for direction. Spacebar for bomb, F for missile.
Airplane PC: Up/Down Arrows for throttle. Left/Right Arrows for yaw. Mouse for pitch/roll.
Mobile Controls: Left joystick for altitude/yaw (throttle/yaw for plane). Right joystick for direction (pitch/roll for plane). Button (B) for bomb, (M) for missile.

Contrôles Mobiles

Gauche: Altitude/Rotation | Droite: Direction

Appuyez sur le bouton ROUGE pour lâcher la bombe

Contrôles PC

Flèches: Altitude/Rotation | Souris: Direction

Clic Droit / Espace: Lâcher la bombe
