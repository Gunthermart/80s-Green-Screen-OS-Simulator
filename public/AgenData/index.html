Mécanisme des Accords (Chords)

Les macros ne sont pas des touches uniques. Vous devez maintenir Alt, puis taper les lettres de la séquence l'une après l'autre. Par exemple, pour insérer "#Important", tapez Alt + H puis I.

Personnalisation

Ouvrez le gestionnaire (Alt+M). Définissez une séquence de lettres et le texte associé. Vos macros sont sauvegardées dans la base de données locale du navigateur.

Le système est optimisé pour une capture rapide suivie d'un tri temporel.

  1. Collecte : Jetez vos notes en vrac dans le brouillon central. Utilisez la voix ou les macros pour structurer.
  2. Tri : Sélectionnez un texte. La barre "VERS" apparaît sous la sélection. Cliquez sur une initiale (L, M, M, J...) pour envoyer le texte vers l'agenda.
  3. Concision : Chaque jour de l'agenda est limité à 6 lignes visibles pour vous forcer à synthétiser l'essentiel.
  4. Indexation : Surlignez les informations critiques (S). La recherche (Ctrl+F) sans texte affichera tous les extraits surlignés.
