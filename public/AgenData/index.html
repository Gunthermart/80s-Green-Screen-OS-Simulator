Knowledge Snippets (Extraits)

L'application est optimisée pour le copier-coller massif. Collez vos textes dans le brouillon, annotez-les ou utilisez le bouton Surligner (S). Un bloc surligné devient une unité de connaissance indexée.

Recherche & Récupération

La recherche (Ctrl+F) est votre point d'accès central. Si le champ est vide, elle affiche tous les blocs surlignés ou annotés de votre base. Un bouton Copier Snippet permet de récupérer le contenu brut instantanément.

Liens Dynamiques

En mode Navigation (Alt+V), les dates (format JJ/MM) et les #tags deviennent cliquables. Cliquer sur une date vous propulse vers ce jour dans l'agenda. Cliquer sur un tag lance une recherche globale.

Transfert Intelligent

Sélectionnez n'importe quel texte dans le brouillon : une barre flottante apparaît. Cliquez sur une initiale (L, M, M, J, V, S, D) pour transférer la sélection vers ce jour de la semaine active.
